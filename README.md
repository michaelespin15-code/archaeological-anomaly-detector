# archaeological-anomaly-detector
Satellite imagery pipeline for archaeological anomaly detection using Google Earth Engine and Sentinel-2 NDVI data. Identifies candidate sites across large terrain areas via z-score analysis.

# Archaeological Anomaly Detection Pipeline

Satellite imagery anomaly detection pipeline built on 
Google Earth Engine using Sentinel-2 NDVI data.

Identified 20 candidate anomaly sites across 49,284 pixels 
in the Ucayali region of Peru using z-score deviation analysis.
Candidate coordinates saved in candidate_sites.csv.

Built by Michael Spin, April 2026.
Next step: cross-reference against known sites and
